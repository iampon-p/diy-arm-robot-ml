Real-time face tracking in Python — 2024

Face detection and tracking via webcam using OpenCV.

Built this as a foundation for computer vision integration with the arm robot project — the plan is to connect it so the arm responds to human presence and movement. Optimised for low latency on standard hardware.

Perception is the missing layer in most DIY robotics. This is where I started building it.
