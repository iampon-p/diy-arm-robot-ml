3D-printed robot arm with ML control — Jan 2025

When you cannot afford a ready-made robot arm, you build one.

Used the UTCC 3D printer lab to print each component, sourced open-source part designs, assembled the arm by hand, then built a Python ML pipeline to control it via camera input. The setup includes a simulation environment, conda-managed reproducible training, and OpenCV-based real-time vision.

Building from physical constraints — print tolerance, motor torque, balance — taught me more about real engineering than coursework alone.

Repo: github.com/iampon-p/diy-arm-robot-ml
